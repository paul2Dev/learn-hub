Stack (Stivă)

Structura LIFO — Last In, First Out. Esențială pentru call stack, undo/redo, parsarea expresiilor și navigarea browserului.

Principiul LIFO

Un Stack funcționează ca o stivă de farfurii: adaugi deasupra, iei de deasupra. Ultimul element adăugat este primul care iese — Last In, First Out. 

Stiva vizualizată:

push(1) → [ 1 ]
push(2) → [ 1, 2 ]
push(3) → [ 1, 2, 3 ]  ← top
pop()   → returnează 3, stiva = [ 1, 2 ]
pop()   → returnează 2, stiva = [ 1 ]
peek()  → returnează 1, stiva = [ 1 ]  (fără să scoată)

Operațiile unui stack:

  • push(val) — adaugă un element deasupra — O(1)
  • pop() — scoate și returnează elementul de deasupra — O(1)
  • peek() — citește vârful fără a-l scoate — O(1)
  • isEmpty() — verifică dacă stiva e goală — O(1)

Implementare în PHP

class Stack {
    private array $items = [];

    public function push(mixed $value): void {
        $this->items[] = $value;
    }

    public function pop(): mixed {
        if ($this->isEmpty()) {
            throw new UnderflowException('Stack is empty');
        }
        return array_pop($this->items);
    }

    public function peek(): mixed {
        if ($this->isEmpty()) {
            throw new UnderflowException('Stack is empty');
        }
        return end($this->items);
    }

    public function isEmpty(): bool {
        return empty($this->items);
    }

    public function size(): int {
        return count($this->items);
    }
}

// Utilizare
$stack = new Stack();
$stack->push(10);
$stack->push(20);
$stack->push(30);

echo $stack->peek();  // 30
echo $stack->pop();   // 30
echo $stack->pop();   // 20
echo $stack->size();  // 1

Implementare în JavaScript

class Stack {
    #items = [];

    push(value) {
        this.#items.push(value);
    }

    pop() {
        if (this.isEmpty()) throw new Error('Stack is empty');
        return this.#items.pop();
    }

    peek() {
        if (this.isEmpty()) throw new Error('Stack is empty');
        return this.#items.at(-1);
    }

    isEmpty() {
        return this.#items.length === 0;
    }

    get size() {
        return this.#items.length;
    }
}

// Utilizare
const stack = new Stack();
stack.push(10);
stack.push(20);
stack.push(30);

console.log(stack.peek());  // 30
console.log(stack.pop());   // 30
console.log(stack.size);    // 2

Aplicații reale

1. Undo / Redo

class TextEditor {
    private string $text = '';
    private Stack $history;

    public function __construct() {
        $this->history = new Stack();
    }

    public function type(string $chars): void {
        $this->history->push($this->text);  // salvăm starea anterioară
        $this->text .= $chars;
    }

    public function undo(): void {
        if (!$this->history->isEmpty()) {
            $this->text = $this->history->pop();  // revenim la starea anterioară
        }
    }

    public function getText(): string { return $this->text; }
}

$editor = new TextEditor();
$editor->type('Hello');
$editor->type(' World');
echo $editor->getText();   // Hello World
$editor->undo();
echo $editor->getText();   // Hello

2. Verificare paranteze echilibrate

function isBalanced(string $str): bool {
    $stack = new Stack();
    $pairs = [')' => '(', ']' => '[', '}' => '{'];

    for ($i = 0; $i < strlen($str); $i++) {
        $ch = $str[$i];
        if (in_array($ch, ['(', '[', '{'])) {
            $stack->push($ch);
        } elseif (isset($pairs[$ch])) {
            if ($stack->isEmpty() || $stack->pop() !== $pairs[$ch]) {
                return false;
            }
        }
    }
    return $stack->isEmpty();
}

var_dump(isBalanced('({[]})'));  // true
var_dump(isBalanced('({[})'));   // false

3. Call Stack — ce este în realitate

// PHP/JS engine-ul tău folosește un stack intern pentru apeluri de funcții:
function a() { b(); }    // push frame a
function b() { c(); }    // push frame b
function c() {           // push frame c
    // stack: [a, b, c]  ← top
}                        // pop c → stack: [a, b]
                         // pop b → stack: [a]
                         // pop a → stack: []
// Stack Overflow = stack-ul s-a umplut (recursivitate infinită)

Verifică-ți înțelegerea

Întrebarea 1 din 5 Ce înseamnă LIFO în contextul unui Stack?
Corect. LIFO înseamnă că elementele se extrag în ordine inversă față de ordinea inserării. Dacă adaugi A, B, C — vei scoate C, B, A. Gândește-te la o stivă de farfurii.
Întrebarea 2 din 5 Care este diferența dintre pop() și peek()?
Corect. peek() (sau top()) este o operație non-destructivă — citești vârful fără a-l elimina din stack. pop() elimină și returnează elementul. Util când vrei să verifici vârful înainte de a decide dacă să-l scoți.
Întrebarea 3 din 5 Ce este un "Stack Overflow" în contextul programării?
Corect. Engine-ul PHP/JS menține un call stack pentru apeluri de funcții. La recursivitate infinită (sau prea adâncă), stack-ul se umple și apare eroarea "Maximum call stack size exceeded" (JS) sau "Maximum function nesting level" (PHP).
Întrebarea 4 din 5 De ce un stack este potrivit pentru funcționalitatea Undo?
Corect. Undo funcționează exact ca un stack: faci acțiuni în ordine (push state), iar când apeși Ctrl+Z, anulez ultima acțiune (pop). LIFO corespunde perfect cu semantica "anulează ce ai făcut ultima dată".
Întrebarea 5 din 5 Care metodă PHP implementează operația pop() a unui stack bazat pe array?
Corect. array_pop() scoate și returnează ultimul element — exact operația pop a unui stack. Combinat cu $arr[] = $val sau array_push() pentru push, un array PHP devine un stack funcțional complet.
