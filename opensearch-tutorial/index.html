OpenSearch cu Laravel

De la zero la integrare completă — 9 lecții, cod concret, quiz-uri interactive

Lecții

01

Ce este OpenSearch? concepte

Arhitectura, cluster, index, document, mapping, shard. Rolul OpenSearch față de MongoDB în proiectul tău.

 02

Instalare și conectare din Laravel cod

Docker Compose pentru OpenSearch, pachete PHP, configurare .env, Service class, comandă Artisan de ping.

 03

CRUD — indexare, citire, actualizare, ștergere cod

Creare index, indexare document, get, partial update, delete, bulk indexing. Pattern MongoDB + OpenSearch.

 04

Căutare full-text — Query DSL cod

match, multi_match cu boost, match_phrase, fuzziness, structura răspunsului, extractHits helper.

 05

Mapping — schema indexului tău concepte

text vs keyword, tipuri de câmpuri, multi-fields, mapping complet pentru articles, comandă Artisan migrate.

 06

Filtre, sortare și paginație cod

bool query (must/filter/should/must_not), term, terms, range, sortare, paginare from/size, query complet parametrizabil.

 07

Integrare Eloquent cu pdphilip/opensearch cod

Modele Eloquent pe OpenSearch, CRUD, whereMatch, paginate, Eloquent Observer pentru sincronizare MongoDB → OpenSearch.

 08

Highlight — evidențierea termenilor căutați cod

pre_tags/post_tags, fragment_size, number_of_fragments, Blade template cu {!! !!}, CSS pentru stilizarea match-urilor.

 09

Agregări — statistici și filtre laterale cod

terms, date_histogram, range aggregation. Pattern faceted search pentru sidebar-uri cu contoare. Combini query + highlight + agregări într-un singur request.
