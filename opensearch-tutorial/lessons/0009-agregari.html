Agregări — statistici din rezultate

terms, range, date_histogram — filtre laterale, contoare, grafice

1. Ce sunt agregările și de ce le vrei

Agregările sunt calcule statistice pe setul de rezultate. Cel mai vizibil exemplu: filtrele laterale din e-commerce.

Taguri

php 47
laravel 31
javascript 18
mysql 12

Autor

Paul 23
Ana 15
Mihai 9

An publicare

2025 34
2024 28
2023 14

Toate aceste numere vin dintr-un singur request OpenSearch care combină query (ce cauți) cu aggregations (statistici despre rezultate).

Diferența față de GROUP BY din SQL SELECT tag, COUNT(*) FROM articles GROUP BY tag în SQL → terms aggregation în OpenSearch.
Dar OpenSearch aplică agregarea pe rezultatele filtrului curent — dacă utilizatorul filtrează după „php", numerele din celelalte facete reflectă automat doar articolele cu „php".

2. Structura unui request cu agregări

Agregările se adaugă în câmpul aggs (sau aggregations), paralel cu query. Poți combina oricâte agregări într-un singur request. 

$response = $this->client->search([
    'index' => 'articles',
    'body'  => [
        'size'  => 10,         // documentele efective (0 dacă vrei doar statistici)

        'query' => [ ... ],   // query normal din Lecțiile 4-6

        'aggs'  => [           // agregările — calculele statistice
            'per_tag'  => [ ... ],   // tu dai un nume fiecărei agregări
            'per_autor' => [ ... ],
            'pe_luna'  => [ ... ],
        ],
    ],
]);

3. terms — count per valoare de câmp

Cea mai folosită agregare. Numără câte documente există per valoare unică a unui câmp. Câmpul trebuie să fie keyword (nu text). 

public function searchWithFacets(string $query, array $filters = []): array
{
    // Filtrele active (din URL: ?tag=php&autor=Paul)
    $filterClauses = [['term' => ['status' => 'published']]];
    foreach ($filters as $field => $value) {
        $filterClauses[] = ['term' => [$field => $value]];
    }

    $response = $this->client->search([
        'index' => 'articles',
        'body'  => [
            'size'  => 10,
            'query' => ['bool' => [
                'must'   => [['multi_match' => [
                    'query'  => $query,
                    'fields' => ['titlu^3', 'continut'],
                ]]],
                'filter' => $filterClauses,
            ]],

            'aggs' => [

                'per_tag' => [
                    'terms' => [
                        'field' => 'taguri',  // trebuie keyword
                        'size'  => 10,       // max 10 taguri returnate
                    ],
                ],

                'per_autor' => [
                    'terms' => [
                        'field' => 'autor.keyword',  // sub-câmpul keyword
                        'size'  => 10,
                    ],
                ],

            ],
        ],
    ]);

    return $this->extractWithAggs($response);
}

Structura răspunsului pentru agregări

// $response['aggregations'] conține rezultatele:
[
    'aggregations' => [
        'per_tag' => [
            'buckets' => [
                ['key' => 'php',        'doc_count' => 47],
                ['key' => 'laravel',    'doc_count' => 31],
                ['key' => 'javascript', 'doc_count' => 18],
            ],
        ],
        'per_autor' => [
            'buckets' => [
                ['key' => 'Paul',  'doc_count' => 23],
                ['key' => 'Ana',   'doc_count' => 15],
            ],
        ],
    ],
]
protected function extractWithAggs(array $response): array
{
    // Extrage bucket-urile într-un format simplu [label => count]
    $agg = fn(string $name) => collect($response['aggregations'][$name]['buckets'] ?? [])
        ->mapWithKeys(fn($b) => [$b['key'] => $b['doc_count']])
        ->toArray();

    return [
        'total'   => $response['hits']['total']['value'],
        'results' => collect($response['hits']['hits'])
            ->map(fn($h) => array_merge(['_id' => $h['_id']], $h['_source']))->toArray(),
        'facets'  => [
            'taguri'  => $agg('per_tag'),    // ['php' => 47, 'laravel' => 31]
            'autori'  => $agg('per_autor'),  // ['Paul' => 23, 'Ana' => 15]
        ],
    ];
}

// Utilizare în controller:
$data = $search->searchWithFacets('laravel');
// $data['facets']['taguri']  = ['php' => 47, 'laravel' => 31, ...]
// $data['facets']['autori']  = ['Paul' => 23, 'Ana' => 15, ...]

4. date_histogram — distribuție în timp

Util pentru grafice de tip „articole publicate per lună" sau filtrare cronologică. 

'aggs' => [

    'pe_luna' => [
        'date_histogram' => [
            'field'             => 'publicat_la',
            'calendar_interval' => 'month',  // 'day', 'week', 'month', 'year'
            'format'            => 'yyyy-MM', // formatul cheii returnate
            'min_doc_count'     => 1,          // ascunde lunile fără articole
        ],
    ],

    'pe_an' => [
        'date_histogram' => [
            'field'             => 'publicat_la',
            'calendar_interval' => 'year',
            'format'            => 'yyyy',
        ],
    ],

],
// Răspuns pentru 'pe_luna':
[
    'buckets' => [
        ['key_as_string' => '2024-11', 'key' => 1730419200000, 'doc_count' => 8],
        ['key_as_string' => '2024-12', 'key' => 1733011200000, 'doc_count' => 12],
        ['key_as_string' => '2025-01', 'key' => 1735689600000, 'doc_count' => 19],
    ],
]

// Extras pentru grafic:
$lunare = collect($response['aggregations']['pe_luna']['buckets'])
    ->mapWithKeys(fn($b) => [$b['key_as_string'] => $b['doc_count']])
    ->toArray();
// ['2024-11' => 8, '2024-12' => 12, '2025-01' => 19]

6. Request complet — search + highlight + agregări

Poți combina tot ce ai învățat în Lecțiile 4–9 într-un singur request:

public function searchComplete(array $params): array
{
    $query   = $params['q']      ?? '';
    $tag     = $params['tag']    ?? null;
    $page    = $params['page']   ?? 1;
    $perPage = 10;

    $filters = [['term' => ['status' => 'published']]];
    if ($tag) $filters[] = ['term' => ['taguri' => $tag]];

    $response = $this->client->search([
        'index' => 'articles',
        'body'  => [
            'size'  => $perPage,
            'from'  => ($page - 1) * $perPage,

            // 1. Query + filtre
            'query' => ['bool' => [
                'must'   => $query
                    ? [['multi_match' => ['query' => $query, 'fields' => ['titlu^3', 'continut'], 'fuzziness' => 'AUTO']]]
                    : [['match_all' => (object)[]]],
                'filter' => $filters,
            ]],

            // 2. Highlight
            'highlight' => [
                'pre_tags'  => ['<em class="hl">'],
                'post_tags' => ['</em>'],
                'fields'    => [
                    'titlu'    => ['number_of_fragments' => 0],
                    'continut' => ['fragment_size' => 150, 'number_of_fragments' => 2, 'no_match_size' => 100],
                ],
            ],

            // 3. Agregări
            'aggs' => [
                'per_tag'  => ['terms' => ['field' => 'taguri',         'size' => 10]],
                'per_autor' => ['terms' => ['field' => 'autor.keyword',  'size' => 10]],
                'pe_an'    => ['date_histogram' => ['field' => 'publicat_la', 'calendar_interval' => 'year', 'format' => 'yyyy', 'min_doc_count' => 1]],
            ],

            'sort' => [['_score' => ['order' => 'desc']]],
        ],
    ]);

    $agg = fn(string $k) => collect($response['aggregations'][$k]['buckets'])
        ->mapWithKeys(fn($b) => [($b['key_as_string'] ?? $b['key']) => $b['doc_count']])->toArray();

    $total = $response['hits']['total']['value'];

    return [
        'total'        => $total,
        'current_page' => $page,
        'last_page'    => (int) ceil($total / $perPage),
        'results'      => collect($response['hits']['hits'])->map(fn($h) => [
            '_id'            => $h['_id'],
            ...$h['_source'],
            'titlu_display'  => $h['highlight']['titlu'][0] ?? $h['_source']['titlu'],
            'fragmente'      => $h['highlight']['continut'] ?? [],
        ])->toArray(),
        'facets' => [
            'taguri'  => $agg('per_tag'),
            'autori'  => $agg('per_autor'),
            'ani'     => $agg('pe_an'),
        ],
    ];
}
