Cum a fost făcut UpNext

Un tur ghidat prin codul real al aplicației UpNext (D:\projects\movies) — un discovery app pentru filme/seriale construit cu Nuxt 4, Nuxt UI, Supabase și TMDB. Fiecare lecție pornește de la fișiere reale din aplicație, nu exemple sintetice: routing, componente, composables, server API cu Nitro, autentificare Google OAuth, schema Postgres cu RLS, motorul de recomandări, și cum se extinde aplicația cu o entitate nouă. Scopul: să te simți confortabil să modifici orice parte a acestei aplicații cu Nuxt.
