Git & GitHub Fluency

De la mental model la CI/CD complet — 9 lecții, cod concret, quiz-uri interactive

Lecții

01

The Git Commit Graph concepte

Cum gândește Git intern: commit graph, HEAD, branches ca pointeri, detached HEAD state. Mental modelul care face totul să facă sens.

 02

Merge Conflicts practică

De ce apar conflictele, cum arată markerii de conflict, cum le rezolvi pas cu pas și strategii de prevenire în echipă.

 03

Rebase vs. Merge concepte

Diferențele reale între cele două strategii, când folosești fiecare, interactive rebase și golden rule-ul pe care nu trebuie să-l încalci.

 04

GitHub Actions practică

Anatomia unui workflow YAML: triggers, jobs, steps, runners. Secrets, artifacts, matrix builds. Primul tău workflow funcțional.

 05

GitHub Webhooks practică

Ce sunt webhook-urile, cum funcționează payload-ul, securitate cu HMAC signature, și când alegi webhooks în loc de Actions.

 06

CI/CD Pipelines practică

Pipeline complet de la push la deploy: build, test, lint, environments, approvals manuale, rollback strategy.

 07

Pull Requests practică

Cum scrii un PR bun, code review eficient, draft PRs, branch protection rules, merge strategies și eticheta de echipă.

 08

Workflow Automations practică

Automatizări Git care salvează timp zilnic: hooks, aliases, scripts, GitHub CLI, auto-labeling, stale bot, dependabot.

 09

Real-World Lab laborator

Scenariu complet end-to-end: feature branch → conflict → PR cu CI → code review → merge → deploy automat. Totul la un loc.
