# Git & GitHub — Tutorial în română

Tutorial practic pentru Git și GitHub — de la mental model la CI/CD complet.
Scris în română, axat pe cod concret și scenarii reale de la job.

## Structura cursului

| # | Lecție | Subiecte |
|---|--------|---------|
| 01 | The Git Commit Graph | Cum gândește Git, commit graph, HEAD, branches, detached HEAD |
| 02 | Merge Conflicts | De ce apar conflictele, cum le rezolvi, strategii de prevenire |
| 03 | Rebase vs. Merge | Diferențe, când folosești fiecare, interactive rebase, golden rule |
| 04 | GitHub Actions | Workflow YAML, triggers, jobs, steps, secrets, artifacts |
| 05 | GitHub Webhooks | Evenimente, payload, securitate, integrare cu servicii externe |
| 06 | CI/CD Pipelines | Pipeline complet: build, test, deploy, environments, approvals |
| 07 | Pull Requests | Code review, draft PRs, branch protection, merge strategies |
| 08 | Workflow Automations | Automatizări Git care salvează timp zilnic în echipă |
| 09 | Real-World Lab | Laborator complet cu scenariu real: feature branch → PR → CI → deploy |

## Cum folosești tutorialul

Deschide `index.html` direct în browser — nu necesită server.

```bash
# Windows
start index.html

# macOS
open index.html
```

Fiecare lecție conține:
- Explicații conceptuale cu diagrame vizuale
- Cod și exemple complete, copy-paste ready
- Quiz interactiv cu feedback instant

## Stack acoperit

- **Git** 2.x
- **GitHub** — Actions, Webhooks, Pull Requests, Branch Protection
- **CI/CD** — pipelines reale cu GitHub Actions
- **Workflow automations** — hooks, scripts, aliases utile

## Flux tipic de lucru

```
feature branch  →  Pull Request  →  CI checks  →  Code Review  →  Merge  →  Deploy
```

Același flux pe care îl folosesc echipele profesioniste — învățat pas cu pas.
