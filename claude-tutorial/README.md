# Claude Code — Tutorial în română

Tutorial practic pentru Claude Code și extensia VS Code — de la instalare la productivitate reală.
Scris în română, axat pe fluxul zilnic al unui programator care vrea să lucreze mai rapid.

## Structura cursului

| # | Lecție | Subiecte |
|---|--------|---------|
| 01 | Instalare și Autentificare | Node.js, npm install, autentificare, prima comandă, modul one-shot |
| 02 | Mental Model | Agent vs. chatbot, context window, sesiuni scurte, permission model |
| 03 | Explorarea Codului | Codebase-uri noi, întrebări de explorare, CLAUDE.md ierarhic |
| 04 | Editare și Implementare | Modificarea fișierelor, multi-file features, iterare pe implementare |
| 05 | Debugging | Stack traces, teste eșuate, buguri de logică, logging temporar |
| 06 | Git Workflow | Commit cu mesaje bune, Pull Requests, diff, conflicte de merge |
| 07 | Extensia VS Code | Instalare, panou lateral, selecție cod ca context, scurtături |
| 08 | Productivitate Zilnică | Slash commands, CLAUDE.md avansat, obiceiuri, limitări reale |
| 09 | Claude Skills | Slash commands personalizate, `$ARGUMENTS`, project vs. global |
| 10 | Claude Hooks | PreToolUse / PostToolUse, variabile de mediu, blocare execuție |
| 11 | Claude Agents | Instanțe paralele, background agents, worktrees, agenți built-in |
| 12 | Agentic Workflow | Bucla percepe-planifică-acționează, permission model, prompts eficiente |

## Cum folosești tutorialul

Deschide `index.html` direct în browser — nu necesită server.

```bash
# Windows
start index.html

# macOS
open index.html
```

Fiecare lecție conține:
- Explicații conceptuale ancorate în exemple reale
- Cod și comenzi complete, copy-paste ready
- Quiz interactiv cu feedback instant

## Referințe rapide

- **Comenzi esențiale Claude Code** — CLI, slash commands și patterns de prompting
- **Scurtături VS Code Extension** — toate scurtăturile pentru fluxul zilnic

## Stack acoperit

- **Claude Code** — CLI, modul interactiv și one-shot
- **VS Code Extension** — panoul lateral, selecție cod, Command Palette
- **CLAUDE.md** — configurare globală și per-proiect
- **Git integrat** — commit, PR, diff, merge conflicts via Claude Code

## Flux tipic de lucru

```
claude → explorare codebase → implementare → debugging → commit → PR
```

Același flux pe care îl rulezi zilnic — accelerat cu Claude Code la fiecare pas.
