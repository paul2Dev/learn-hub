Extensia VS Code

Același Claude Code, direct în editorul tău — fără să schimbi contextul între terminal și VS Code.

Instalare

Deschide VS Code și instalează extensia oficial:

  1. Apasă Ctrl+Shift+X (Extensions)
  2. Caută Claude Code — publisher: Anthropic
  3. Click Install
  4. Reload VS Code dacă e cerut

Alternativ, din terminal:

code --install-extension anthropic.claude-code

Autentificare: Extensia folosește aceeași autentificare ca și CLI-ul. Dacă ai rulat deja claude în terminal, ești autentificat automat.

Interfața extensiei

Extensia adaugă un panou lateral Claude Code în VS Code. Îl deschizi din bara de activitate (iconița Claude) sau cu scurtătura Ctrl+Shift+C (Windows/Linux) / Cmd+Shift+C (Mac).

Panoul conține:

  • Un câmp de input pentru mesaje
  • Istoricul conversației curente
  • Buton de reset sesiune
  • Status autentificare

Superputerea extensiei: selecția ca context

Acesta e motivul principal pentru care extensia bate terminalul în fluxul zilnic: poți selecta cod direct din editor și îl trimiți automat ca context.

Cum funcționează:

  1. Selectezi un bloc de cod în editor (o funcție, o clasă, un fișier)
  2. Apesi Ctrl+Shift+C sau click dreapta → "Ask Claude Code"
  3. Scrii întrebarea — codul selectat e deja atașat ca context
# Nu mai trebuie să faci asta:
> Uite funcția: [paste manual de 50 de linii]
   Ce face ea?

# Selectezi funcția, apesi shortcut, și scrii direct:
> Ce face această funcție și ce returnează?

Pattern eficient: Selectează funcția cu bug-ul, apasă shortcut-ul, paste eroarea. Contextul e complet fără nicio copiere manuală.

Scurtături esențiale

Cele mai importante scurtături din extensia VS Code:

  • Ctrl+Shift+C — Deschide panoul / trimite selecția ca context
  • Ctrl+Enter — Trimite mesajul din câmpul de input
  • Esc — Oprește generarea curentă
  • Alt+L — Inserează codul sugerat direct în fișierul activ

Toate scurtăturile sunt vizibile și modificabile în File → Preferences → Keyboard Shortcuts → caută "Claude".

Vezi referința completă în Scurtături VS Code Extension.

Terminal integrat vs. panou lateral

O întrebare frecventă: când folosești panoul lateral și când deschizi terminalul VS Code pentru claude CLI?

  • Panou lateral — Când lucrezi activ în editor și vrei să întrebi despre codul pe care îl vezi. Selecție → context → întrebare.
  • Terminal (CLI) — Când vrei să rulezi comenzi complexe cu mai mulți pași, să execuți scripturi sau să ai o sesiune lungă de implementare. CLI-ul are acces complet la toate uneltele (git, npm, etc.).

Poți folosi ambele simultan — o sesiune CLI în terminalul integrat și panoul lateral pentru întrebări punctuale. Sunt instanțe independente.

Command Palette

Apasă Ctrl+Shift+P și caută "Claude" pentru toate comenzile disponibile:

  • Claude Code: New Session — resetează conversația curentă
  • Claude Code: Open Panel — deschide panoul lateral
  • Claude Code: Explain Selection — explică codul selectat
  • Claude Code: Fix Selection — repară codul selectat
