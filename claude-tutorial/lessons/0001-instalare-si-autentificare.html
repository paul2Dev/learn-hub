Claude Code Tutorial · Lecția 1 din 12

Instalare și Autentificare

De la zero la prima comandă funcțională — în mai puțin de 5 minute.

Ce este Claude Code?

Claude Code este un agent AI în terminal care înțelege proiectul tău: citește fișiere, scrie cod, rulează comenzi și interacționează cu git — totul direct din linia de comandă sau din VS Code.

Diferența față de un chatbot obișnuit: Claude Code acționează în mediul tău de dezvoltare, nu doar vorbește despre el. Mai mult despre asta în Lecția 2.

Cerință: Node.js versiunea 18 sau mai nouă. Verifică cu node --version. Dacă nu ai Node.js, descarcă-l de la nodejs.org.

Instalare

Un singur comandă, globală, din orice terminal:

npm install -g @anthropic-ai/claude-code

Verifică instalarea:

claude --version

Ar trebui să vezi ceva de genul 1.x.x. Dacă primești command not found, înseamnă că directorul global npm nu e în PATH — consultă documentația npm.

Autentificare

Rulează simplu:

claude

La prima pornire, Claude Code îți va cere autentificarea. Ai două variante:

  1. Login cu claude.ai — recomandat dacă ai abonament Claude Pro sau Max. Apasă Enter și se deschide browserul automat.
  2. API Key — dacă ai o cheie Anthropic API. Setează variabila de mediu: ANTHROPIC_API_KEY=sk-ant-...

Recomandare: Dacă ești programator individual, abonamentul Claude Max (cu acces la Claude Code) oferă cel mai bun raport calitate/preț față de plata per token via API.

Prima comandă

Navighează într-un proiect existent și rulează:

cd /calea/spre/proiectul-tau
claude

Acum ești în modul interactiv. Încearcă:

> Ce fișiere are acest proiect și ce face el?

Claude Code va citi structura proiectului și îți va răspunde în context. Pentru a ieși, tastează /exit sau apasă Ctrl+C.

Important: Rulează întotdeauna claude din rădăcina proiectului tău — acolo unde e .git. Asta îi dă contextul corect.

Modul one-shot (fără sesiune interactivă)

Pentru sarcini rapide, poți trimite un prompt direct fără să intri în modul interactiv:

claude "Câte fișiere JavaScript are acest proiect?"

Claude va răspunde și va ieși automat. Util pentru scripturi sau comenzi rapide din CI.

Deschide o sesiune claude și întreabă direct: "Explică-mi ce înseamnă X din această lecție."
