Claude Code Tutorial · Referință

Scurtături VS Code Extension

Toate scurtăturile extensiei Claude Code pentru VS Code — modificabile din File → Preferences → Keyboard Shortcuts.

Panou și navigare

AcțiuneWindows / LinuxMac
Deschide / închide panoul Claude Code Ctrl+Shift+C Cmd+Shift+C
Focusează câmpul de input Ctrl+Shift+C (dacă panoul e deja deschis) Cmd+Shift+C
Deschide Command Palette → Claude Ctrl+Shift+P → tastează "Claude" Cmd+Shift+P → tastează "Claude"

În câmpul de input

AcțiuneWindows / LinuxMac
Trimite mesajul Ctrl+Enter sau Enter Cmd+Enter sau Enter
Linie nouă în mesaj (fără trimitere) Shift+Enter Shift+Enter
Oprește generarea curentă Esc Esc
Mesaj anterior din istoric ↑ (în câmp gol) ↑ (în câmp gol)

Selecție și context

AcțiuneWindows / LinuxMac
Trimite selecția curentă ca context Selectează cod → Ctrl+Shift+C Selectează cod → Cmd+Shift+C
Explică selecția (Command Palette) Ctrl+Shift+P → "Claude: Explain Selection" Cmd+Shift+P → "Claude: Explain Selection"
Repară selecția (Command Palette) Ctrl+Shift+P → "Claude: Fix Selection" Cmd+Shift+P → "Claude: Fix Selection"

Sesiune

AcțiuneUnde
Sesiune nouă (resetează conversația) Buton "New Session" din panoul lateral sau Command Palette → "Claude Code: New Session"
Inserează codul sugerat în editor Alt+L sau buton "Insert" de lângă blocul de cod din răspuns

Tips de utilizare

SituațieAbordare recomandată
Ai o eroare în consolă și vrei să o rezolvi rapid Selectează eroarea din terminal integrat → Ctrl+Shift+C → "Ce cauzează această eroare?"
Vrei să înțelegi o funcție complexă Selectează întreaga funcție → Ctrl+Shift+C → "Explică pas cu pas ce face această funcție"
Vrei să refactorizezi un bloc de cod Selectează blocul → Ctrl+Shift+C → "Refactorizează pentru claritate, păstrând același comportament"
Vrei să adaugi tipuri TypeScript la cod JS Selectează fișierul/funcția → Ctrl+Shift+C → "Adaugă type annotations TypeScript"
