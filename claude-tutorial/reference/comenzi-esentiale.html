Comenzi Esențiale Claude Code

Cheat sheet complet pentru utilizare zilnică — CLI, slash commands și patterns.

CLI — Pornire

ComandăCe face
claudePornește o sesiune interactivă în directorul curent
claude "prompt"Execută un prompt one-shot și iese
claude --versionAfișează versiunea instalată
claude --helpListează toate flag-urile disponibile
claude -p "prompt"Mod non-interactiv (ideal pentru scripturi)

Slash Commands (în sesiunea interactivă)

ComandăCe face
/helpAfișează toate comenzile disponibile
/clearGolește conversația curentă, pornești fresh
/compactComprimă istoricul conversației (eliberează context)
/costAfișează tokene consumate în sesiunea curentă
/statusStarea autentificării și a configurației
/exitÎnchide sesiunea (alternativ: Ctrl+C)

Patterns de prompting — Explorare

PatternCând îl folosești
"Explică-mi structura acestui proiect"Prima zi pe un proiect nou
"Cum funcționează [funcție/modul]?"Înțelegerea codului necunoscut
"Găsește toate locurile unde se apelează [X]"Impact analysis înainte de refactorizare
"Există cod duplicat în proiect?"Audit de calitate a codului
"Creează un CLAUDE.md pentru acest proiect"Configurare inițială a proiectului

Patterns de prompting — Implementare

PatternCând îl folosești
"Adaugă [feature] în [fișier], urmând structura din [exemplu]"Implementare consistentă cu stilul existent
"Implementează [feature] care să facă X, Y, Z"Features cu cerințe clare
"Refactorizează [funcție] pentru a fi mai [eficientă/clară/testabilă]"Îmbunătățire cod existent
"Scrie teste pentru [funcție/endpoint]"Test-driven sau acoperire de teste

Patterns de prompting — Debugging

PatternCând îl folosești
"Primesc această eroare: [stack trace]. Ajută-mă să o rezolv."Erori cu stack trace clar
"Rulează testele și repară ce eșuează"Sesiune de fix tests
"Funcția X returnează [Y] dar ar trebui să returneze [Z] pentru input [W]"Buguri de logică fără erori
"Ce s-a schimbat în ultimele N commit-uri în [modul]?"Bug apărut recent, cauza necunoscută

Patterns de prompting — Git

PatternCând îl folosești
"Fă commit cu modificările curente"Salvarea progresului cu mesaj descriptiv
"Creează un PR pentru branch-ul curent față de main"Finalizare feature, gata de review
"Ce am schimbat față de main?"Revizie înainte de PR
"Rezolvă conflictele de merge păstrând logica din branch-ul meu"Merge conflicts
