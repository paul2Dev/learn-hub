JavaScript/TypeScript: embeddings în Node.js și browser

Aceleași concepte din Lecția 4, dar în JavaScript — cu OpenAI SDK și Transformers.js pentru modele locale.

Setup Node.js

mkdir embeddings-js && cd embeddings-js
npm init -y

# Client OpenAI oficial pentru Node.js
npm install openai

# Transformers.js — modele locale în Node.js sau browser
npm install @xenova/transformers

# Utilitare
npm install dotenv

Creează un fișier .env cu API key-ul: 

OPENAI_API_KEY=sk-proj-...tu-cheia-ta...

Varianta 1: OpenAI SDK (Node.js)

// embed-openai.js
import 'dotenv/config';
import OpenAI from 'openai';

const client = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

async function getEmbedding(text, model = 'text-embedding-3-small') {
  const response = await client.embeddings.create({
    input: text.replace(/\n/g, ' '),
    model,
  });
  return response.data[0].embedding; // float[]
}

// Test
const embedding = await getEmbedding('Pisica mea doarme pe canapea.');
console.log('Dimensiuni:', embedding.length);   // 1536
console.log('Primele 5:', embedding.slice(0, 5));
ESM vs CommonJS
OpenAI SDK v4+ folosește ES Modules. Adaugă "type": "module" în package.json sau folosește extensia .mjs pentru fișiere. Dacă preferi CommonJS, folosește require() și importul const OpenAI = require('openai').default.

Cosine Similarity în JavaScript

JavaScript nu are NumPy, dar formula este simplă de implementat: 

function cosineSimilarity(a, b) {
  if (a.length !== b.length) throw new Error('Vectorii trebuie să aibă aceeași lungime');

  let dot = 0, normA = 0, normB = 0;
  for (let i = 0; i < a.length; i++) {
    dot   += a[i] * b[i];
    normA += a[i] * a[i];
    normB += b[i] * b[i];
  }
  return dot / (Math.sqrt(normA) * Math.sqrt(normB));
}

// Exemplu de utilizare
const embA = await getEmbedding('Pisica mea doarme pe canapea.');
const embB = await getEmbedding('Felina mea se odihnește pe fotoliu.');
const embC = await getEmbedding('Factura de electricitate a crescut.');

console.log('A vs B:', cosineSimilarity(embA, embB).toFixed(4)); // ~0.88
console.log('A vs C:', cosineSimilarity(embA, embC).toFixed(4)); // ~0.15

Semantic Search complet în Node.js

// search.js
import 'dotenv/config';
import OpenAI from 'openai';

const client = new OpenAI({ apiKey: process.env.OPENAI_API_KEY });

function cosineSimilarity(a, b) {
  let dot = 0, normA = 0, normB = 0;
  for (let i = 0; i < a.length; i++) {
    dot += a[i] * b[i]; normA += a[i] ** 2; normB += b[i] ** 2;
  }
  return dot / (Math.sqrt(normA) * Math.sqrt(normB));
}

async function embed(text) {
  const res = await client.embeddings.create({ input: text, model: 'text-embedding-3-small' });
  return res.data[0].embedding;
}

// Corpus
const corpus = [
  'Cum resetez parola contului meu?',
  'Ghid de instalare a aplicației mobile.',
  'Politica de retur și refund pentru comenzi online.',
  'Cum schimb adresa de livrare după plasarea comenzii?',
  'Suport pentru plăți cu cardul în aplicație.',
];

// Indexare — calculat O SINGURĂ DATĂ
const corpusEmbeddings = await Promise.all(corpus.map(embed));

// Funcție de search
async function semanticSearch(query, topK = 3) {
  const queryEmb = await embed(query);
  const scores = corpusEmbeddings.map((docEmb, i) => ({
    score: cosineSimilarity(queryEmb, docEmb),
    doc: corpus[i],
  }));
  return scores.sort((a, b) => b.score - a.score).slice(0, topK);
}

// Test
const results = await semanticSearch('vreau să returnez un produs');
results.forEach(({ score, doc }) => {
  console.log(`[${score.toFixed(3)}] ${doc}`);
});

Rulează cu:

node search.js

Varianta 2: Transformers.js (local, gratuit)

@xenova/transformers descarcă modelul prima oară și îl cachează local. Funcționează atât în Node.js cât și în browser (via WebAssembly). 

// embed-local.js
import { pipeline } from '@xenova/transformers';

// Pipeline-ul se inițializează O SINGURĂ DATĂ
// Prima rulare: descarcă modelul (~90MB) și cachează
const extractor = await pipeline(
  'feature-extraction',
  'Xenova/paraphrase-multilingual-MiniLM-L12-v2'
);

async function getEmbeddingLocal(text) {
  const output = await extractor(text, { pooling: 'mean', normalize: true });
  return Array.from(output.data); // convertim la array obișnuit
}

// Test
const emb = await getEmbeddingLocal('Pisica mea doarme pe canapea.');
console.log('Dimensiuni:', emb.length);   // 384
console.log('Primele 5:', emb.slice(0, 5));
Parametrul normalize: true
Când folosești normalize: true, vectorul este normalizat la lungime unitară. Asta simplifică calculul cosine similarity — devine simplu produs scalar: a.reduce((sum, v, i) => sum + v * b[i], 0). Dacă omiți normalizarea, folosește formula completă cu norma.

TypeScript — tipuri pentru embeddings

// types.ts
export type Embedding = number[];

export interface EmbeddedDocument {
  text: string;
  embedding: Embedding;
}

export interface SearchResult {
  doc: EmbeddedDocument;
  score: number;
}

// embed.ts
import OpenAI from 'openai';
import type { Embedding, EmbeddedDocument, SearchResult } from './types.js';

const client = new OpenAI();

export async function embed(text: string): Promise<Embedding> {
  const res = await client.embeddings.create({
    input: text,
    model: 'text-embedding-3-small',
  });
  return res.data[0].embedding;
}

export function cosineSimilarity(a: Embedding, b: Embedding): number {
  return a.reduce((sum, v, i) => sum + v * b[i], 0) /
    (Math.sqrt(a.reduce((s, v) => s + v * v, 0)) *
     Math.sqrt(b.reduce((s, v) => s + v * v, 0)));
}

export async function semanticSearch(
  query: string,
  index: EmbeddedDocument[],
  topK: number = 3,
): Promise<SearchResult[]> {
  const qEmb = await embed(query);
  return index
    .map(doc => ({ doc, score: cosineSimilarity(qEmb, doc.embedding) }))
    .sort((a, b) => b.score - a.score)
    .slice(0, topK);
}

Verifică-ți înțelegerea

Cinci întrebări. Dă clic pe un răspuns — feedback-ul este imediat.

Întrebarea 1 din 5 Ce returnează client.embeddings.create() din OpenAI SDK pentru Node.js?
Corect. Response-ul are structura { data: [{ embedding: number[], ... }] }. Accesezi vectorul cu response.data[0].embedding — este un simplu array JavaScript de numere.
Întrebarea 2 din 5 Care este avantajul principal al @xenova/transformers față de OpenAI SDK?
Corect. Transformers.js rulează inferența local, în Node.js sau browser, via WebAssembly sau WebGPU. Zero cost API, zero date în cloud — ideal pentru date sensibile sau aplicații offline.
Întrebarea 3 din 5 Dacă folosești normalize: true la Transformers.js, cum simplifici calculul cosine similarity?
Corect. Când vectorii sunt normalizați (lungime 1), cosine similarity = dot product = a.reduce((sum, v, i) => sum + v * b[i], 0). Nu mai calculezi normele. Asta e o optimizare frecventă în vector databases.
Întrebarea 4 din 5 Cum vectorizezi eficient 100 de documente simultan în JavaScript (fără să faci call-uri secvențial)?
Corect. Promise.all(corpus.map(embed)) lansează toate request-urile simultan și așteaptă finalizarea tuturor. Un for...of cu await ar face call-urile secvențial — de ~100x mai lent.
Întrebarea 5 din 5 Un user din România trimite un query în română, dar corpus-ul tău de documente este în engleză. Ce se întâmplă la semantic search cu OpenAI text-embedding-3-small?
Corect. Modelele OpenAI de embedding sunt antrenate multilingv — aceeași idee exprimată în română și engleză produce vectori apropiați în spațiu. Nu ai nevoie de traducere. Sentence Transformers multilingual are același comportament.
